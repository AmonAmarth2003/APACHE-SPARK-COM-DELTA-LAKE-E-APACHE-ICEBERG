# Apache Spark (PySpark)

O Apache Spark é um framework de processamento distribuído projetado para
processar grandes volumes de dados de forma paralela.

## PySpark

PySpark é a interface Python para o Apache Spark, permitindo a criação de
pipelines de dados e consultas SQL sobre grandes volumes de dados.

Neste projeto, o Spark é utilizado como engine de processamento para operar
sobre tabelas Delta Lake e Apache Iceberg.
